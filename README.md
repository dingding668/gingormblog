# gingormblog
这个是我学习gin+gorm编写的一个小项目

目录结构
   
go.mod           // 项目依赖<br>
main.go          //主程序<br>
api         <br>
config           // 项目配置入口   <br>
log              // 项目日志<br>
middleware       // 中间件<br>
model            // 数据模型层<br>
routes/router.go // 路由入口    <br>
utils            // 项目公用工具库<br>


项目功能

用户管理权限设置<br>
用户密码加密存储<br>
文章分类自定义<br>
列表分页<br>
图片上传七牛云<br>
JWT认证<br>
自定义日志功能<br>
跨域cors设置<br>
